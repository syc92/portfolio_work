Balls!

This is a course project of mine for an AR/VR development class. This project uses A-Frame, and it’s built on Glitch.

Year: 2020
Project type: Course project
Duration: 1.5 days
Role: VR Developer, Designer

Overview

Insert objects

When the user clicks the “B button”, a planet (shape) will drop into the box, and the planet position is randomly assigned within the area of the box. The color of the planet is also randomly generated.

Switch shapes with menu

When the user clicks on the left hand trigger, the menu attached to the left hand will show up. The user can then use the raycaster, hovering on the items in the menu to change the shape of the planet.

Manipulate objects

All the objects inserted into the scene are grabbable, droppable, and droppable. You can use the super “right” hand to interact with those planets.
