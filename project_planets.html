Planets!

This is a course project of mine for an AR/VR development class. This project uses Unity with ARKit, ARFoundation, and is deployed to a iOS device. (Acknowledgement: It is based on an example provided by Kangning Chen.)

Year: 2020
Project type: Course project
Duration: 1.5 days
Role: AR Developer, Designer

Overview

How it works:

  • Move your device to find a plane.
  • When a plane is detected, a “Place button” will be shown, and the text hint will change to:
    “Click to Place to add object.
    Click object to resize and rotate.
    Double click object to remove.”
  • Place object: Press the Place button to add an object to the scene. (The objects will be added where the placement indicator was.)
  • Remove object: Double click to remove any objects on the screen.
  • Resize and rotate: Single click to select an object, and use two fingers to resize and rotate.
